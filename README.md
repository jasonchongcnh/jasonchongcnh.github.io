- **Location**: `Map.html`

- **Features**:
    - **Glassmorphism UI**: Floating panels with blur effects for a modern look.
    - **Geolocation**: High-accuracy positioning to locate the user.

    - **Dark/Light Mode**: Optimized for visual clarity.
